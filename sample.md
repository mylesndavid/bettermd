# Welcome to BetterMD

A simple markdown viewer for macOS. Like Preview, but for **markdown**.

## Features

- **Rendered view** — beautiful formatted markdown
- **Raw view** — see the source, toggle with `⌘⇧R`
- **Save as PDF** — the simplest md→pdf in the west (`⌘⇧E`)
- **Live reload** — edit your file, see changes instantly
- **Drag & drop** — just drop a `.md` file onto the window
